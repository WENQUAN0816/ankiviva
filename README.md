# ankiviva

Anki decks for PhD viva preparation and thesis revision.

## Decks

- `decks/论文知识点+高频问题问答.tsv`
  Combined deck of thesis knowledge points, high-frequency viva questions, and answering principles.
- `decks/VIVA防守问题+论文信息.tsv`
  Full viva deck including defense questions, thesis information, oral transitions, and presentation structure.
- `decks/thesis_academic_words_phrases_merged.tsv`
  Combined academic vocabulary, phrases, and collocations from the thesis.
- `decks/thesis_defense_anki_terms.tsv`
  Simplified thesis term deck without type labels.
- `decks/thesis_defense_anki_terms_with_type.tsv`
  Thesis term deck with type labels (`word`, `phrase`, `collocation`).

CSV versions are included for the same decks.
